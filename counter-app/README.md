# Counter App

A simple and interactive Counter Application built using **HTML, CSS, and JavaScript**.  
Users can increase, decrease, and reset the counter value with a clean and modern UI.

## Features

- Increase counter value
- Decrease counter value
- Reset counter to zero
- Responsive UI
- Glassmorphism design with gradient background
- Smooth button hover animations

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)


## How to Run

1. Clone the repository:

2. Open the project folder

3. Run `counter.html` in your browser

## How It Works

- Click **+** button to increase the counter
- Click **−** button to decrease the counter
- Click **⟲** button to reset the counter to zero
