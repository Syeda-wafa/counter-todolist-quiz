# Quiz

A simple interactive Quiz Application built using **HTML, CSS, and JavaScript**.  
Users can select a subject, choose difficulty level, answer multiple-choice questions, and view their final score.

## Features

- Multiple subjects:
  - HTML
  - CSS
  - JavaScript
  - React
  - Python
  - SQL
- Three difficulty levels:
  - Easy
  - Intermediate
  - Hard
- 15-second timer for each question
- Score tracking
- Progress bar
- Correct/Wrong answer highlighting
- Restart quiz option

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

1. Clone the repository:

2. Open project folder

## How It Works

1. Select a subject
2. Choose difficulty level
3. Answer quiz questions before timer ends
4. Get final score after completion
