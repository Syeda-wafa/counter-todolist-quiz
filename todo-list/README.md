# To-Do App

A simple and user-friendly To-Do Application built using **HTML, CSS, and JavaScript**.  
This app helps users manage daily tasks by adding, editing, and deleting tasks with priority levels and due dates.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Task counter
- Set task priority (Low / Medium / High)
- Assign due dates
- Clean glassmorphism UI
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## How to Run

1. Clone the repository:

2. Open the project folder

3. Run `todo.html` in your browser

## How It Works

1. Enter task name
2. Add description (optional)
3. Select task priority
4. Choose due date
5. Click **Add Task**
6. Use **Edit** or **Delete** buttons to manage tasks
