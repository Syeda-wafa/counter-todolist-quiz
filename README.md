# Frontend Projects Collection

This repository contains three frontend web development projects built using HTML, CSS, and JavaScript.

## Projects Included

### 1. Counter App
A simple counter application with increment, decrement, and reset functionality.

**Features**
- Increase counter
- Decrease counter
- Reset counter

**Tech Stack**
- HTML
- CSS
- JavaScript

---

### 2. To-Do App
A task management application to add, edit, and delete daily tasks.

**Features**
- Add tasks
- Edit tasks
- Delete tasks
- Task counter
- Priority selection
- Due date

**Tech Stack**
- HTML
- CSS
- JavaScript

---

### 3. Quiz App
An interactive quiz application with multiple subjects and difficulty levels.

**Features**
- Multiple subjects
- Difficulty levels
- Timer
- Score tracking
- Progress bar

**Tech Stack**
- HTML
- CSS
- JavaScript
